SICOPE Model
=========
Model Based Testing tool using Single Color Petrinet Model.

Usage
--------------------

```
mkdir sicope-model
cd sicope-model
wget https://raw.githubusercontent.com/sicope-model/sicope-model/master/docker-compose.yaml
docker-compose up
```
